# data_engineer_lebedev_arseniy
### Инструкция по запуску
- Открываем директорию, в которую поместим работу
- git clone https://github.com/Faravells/data_engineer_lebedev_arseniy
- cd dvfu-data-analyst
- При надобности изменить период расчета агрегации, свои значения можно вписать в data-aggregator/date-interval.txt
- docker-compose up
- После запуска и отработки скриптов, выходные данные будут в файле output.csv

### Результаты работы
В результате работы была спроектирована база данных для хранения пользовательских логов форума, создан Docker контейнер с MySql базой данных, база данных была заполнена данными, сгенерированными в скрипте data-generator.py, сгенерированные данные были взяты из поднятой базы данных скриптом data-aggregator.py и на их основе была посчитана статистика, выведенная в output.csv.
